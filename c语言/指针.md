##### 指针的接收保存
```c
   int a[10];
   int b[5][6];
   //一维数组用一个普通指针保存没问题
   int *p1 = a;
   //int **p2 = b;incompatible pointer type
   //不正确的指针类型
   //二维数组必须要用下面这种方式保存
   int (*p2)[6] = b;
```
指针只在乎初始位置和数据宽度  
换到二维数组上来说  
就是  
二维数组的初始地址和一行元素的总宽度(列长)
